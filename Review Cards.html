CSS Transitions & Animations Review

What are transitions and animations used for?
To implement more visual activity and interest to a web page.
transition-property
Specifies which CSS property will animate.
transition-duration
Sets how long the transition takes to complete.
What does "ease-in-out" do?
Starts and ends slow, but is faster in the middle.
animation-name
Connects a keyframes rule to an element.
animation-iteration-count
Sets how many times the animation will repeat.
Difference between transition & animation?
Transitions go from one state to another, animations have multiple steps.
What does @keyframes do?
Defines stages of a CSS animation.
